Converting and outputting numeric types

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a Kotlin program that reads a single string containing an integer or floating-point number. Your task is to check the type of the number, convert it to the other type and output it. If the input number is an integer, convert it to a floating-point number, keeping the same numeric value (decimal part should be '.0'). Meanwhile, if the input number is a floating-point number, convert it to the nearest rounded integer.
Sample Input:
42

Sample Output:
42.0


Sample Input:
3.14

Sample Output:
3



Memory limit: 256 MB
Time limit: 8 seconds
