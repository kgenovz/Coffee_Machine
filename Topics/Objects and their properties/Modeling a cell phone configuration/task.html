Modeling a cell phone configuration

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You need to model a basic configuration of a cell phone. The cell phone has a 'name' and 'version'. The 'name' refers to the brand name of the cell phone (e.g., 'Samsung'), and 'version' refers to its version (e.g., 'S10'). Write a program in Kotlin that takes the name and version as input (on two separate lines) and creates a CellPhone object. The CellPhone object should have two properties: 'brandName' and 'version'. The program should then print the details of the cell phone in the following format: 'The cell phone is a {brandName} with version {version}'.
Sample Input:
Samsung
S10

Sample Output:
The cell phone is a Samsung with version S10


Sample Input:
Apple
12

Sample Output:
The cell phone is a Apple with version 12



Memory limit: 256 MB
Time limit: 8 seconds
